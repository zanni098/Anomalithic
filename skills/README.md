# Built-in skills

Skills that ship with the Anomalithic runtime. Each folder contains a Claude
`SKILL.md` (and optional `reference/` files) discovered by `@anomalithic/skills`.
Run `anomalithic skills` from anywhere and these appear out of the box.

| Skill | What it teaches |
|---|---|
| [`loop-engineering`](./loop-engineering) | Stop prompting — write autonomous loops that verify themselves. The `/goal` feature, the 5-step loop, maker–checker verification, stop/cost controls, and the 24/7 system. |

Drop a new folder with a `SKILL.md` here to add another built-in skill.
